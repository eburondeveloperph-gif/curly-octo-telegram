# Translator

Real-time voice translation powered by Gemini Live, with a custom animated entry screen, persistent local history, and optional Supabase-backed session storage.

## Run locally

Prerequisite: Node.js

1. Install dependencies: `npm install`
2. Set `GEMINI_API_KEY` in [.env.local](.env.local)
3. Start the dev server: `npm run dev`

## Checks

- Build: `npm run build`
- Typecheck: `npm run typecheck`
